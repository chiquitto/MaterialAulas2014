==

O componente Zend_Filter

O componente Zend_Filter prove um conjunto de filtros de dados que geralmente são necessários.
O componente também fornece um mecanismo de encadeamento simples de vários filtros que podem ser aplicados à mesma informação.

==

O que é um filtro ?

No mundo real, um filtro geralmente é utilizado para remover partes desnecessárias de algo.
Por exemplo, quando estamos preparando café, utilizamos um filtro para remover o pó, para então utilizarmos o que sai do filtro.

Os filtros, dentro no nosso contexto, serão utilizados para filtrar informações fornecidas à nossa aplicação.

==

Uso básico dos componente no Zend_Filter

Todos os filtros do Zend_Filter implementam a interface Zend_Filter_Interface, que define um único método: o filter().

<?php
$htmlEntities = new Zend_Filter_HtmlEntities();
echo $htmlEntities->filter('&'); // &amp;
echo $htmlEntities->filter('"'); // &quot;
?>
